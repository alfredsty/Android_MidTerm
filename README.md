## 설명
* 4갸의 과목점수를 입력하고 평균값을 구한다.
* 시험 평균점수에 따라 성적산출에서 결과가 달라짐
* 4개의 과목점수를 모두 입력 안 하고 성적산출 시 경고메세지가 뜸.
